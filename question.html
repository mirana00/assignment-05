Important FAQ

What is the document object model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML document as a tree of objects, where each element is a node that can be accessed, modified, or deleted by JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, these methods can be used, 1. document.getElementById() 2. document.getElementsByClassName() 3. document.getElementsByTagName() 4. document.querySelector() 5. document.querySelectorAll().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element rather than individual child elements. The event listener captures events from child elements as they "bubble up" through the DOM. Itâ€™s useful because it saves memory by reducing the number of event listeners, and dynamically added elements are automatically handled.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using the DOM with setAttribute() or by directly changing properties like element.src for images or element.href for links. For styles, you can use the style property, like element.style.color = 'red'; to change an element's color.
